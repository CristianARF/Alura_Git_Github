Git

Esta es una página que nos permite aprender a subir archivos a Git

A continuación algunas de las funciones de Git

  • Llevar registro de los cambios en archivos de un repositorio.
  • Coordinar el trabajo que varias personas realizan sobre archivos compartidos en un repositorio de código.
  • Proporcionar un listado de los cambios(commits) a los que podemos volver atrás en el tiempo en cualquiera de esos cambios o commits.

GitHub

¿Qué es GitHub?

GitHub es una plataforma de desarrollo colaborativo para alojar proyectos utilizando el sistema de control de versiones Git. Es un servicio en línea al que los desarrolladores que utilizan Git pueden conectarse y cargar o descargar recursos.

Caracteristicas:

  • Máximo 3 colaboradores gratis.
  • Ofrece 500Mb de almacenamiento gratuito.
  • Cada usuario tiene su perfil en el que aparecen sus contribuciones, y los usuarios pueden seguirse entre si para ver su actividad.
